# Headphone_Scraper_Project
A program written in python using scrapy and Zyte-API to crawl headphone details.

### Headphone Crawler Program



##### Developed by Hakan Cetinkaya



###### Required installments:

* VS Code
* scrapy-playwright
* python
* scrapy
* Zyte API





###### To run the program:

Install Web Scraper Copilot extension in VS Code.

Open settings.py and enter your Zyte API key.

Open sidebar view for Web Scraper Copilot, and go to spiders.

Run the headphones\_crawler spider and specify the output file as a .csv file.
